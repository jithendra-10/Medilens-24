# HealthBridge (MediLens AI)

**HealthBridge** is a premium AI-powered medical command center that transforms complex lab reports into clear, actionable health intelligence for the entire family.

## Core Features
- **Overview (Dashboard)**: Central command center for health alerts and recent activities.
- **Medical History**: Secure digital vault for laboratory reports and historical trends.
- **Family Hub**: Multi-patient management system for organized family health tracking.
- **AI Assistant**: RAG-powered intelligent medical companion with General and Contextual modes.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS v4, Framer Motion, Lucide Icons.
- **Backend**: FastAPI, Python, SQLAlchemy.
- **AI**: Google Gemini Pro (Neural Engine).
